#+TITLE: kenchu dotfiles
* Dotfiles
This is my dotfiles.

* Requirements
** Stow
To install [[https://www.gnu.org/software/stow/][stow]].
#+begin_src zsh
brew install stow
#+end_src
** Subversion
Zinit needs subversion to download packages.
#+begin_src zsh
brew install subversion
#+end_src

* Getting Start 
To symlink the ssh config.
#+begin_src zsh
stow -S ssh
#+end_src

To remove the symlink,
#+begin_src zsh
stow -D ssh
#+end_src

To re-stow,
#+begin_src zsh
stow -R ssh
#+end_src
